# Enterprise Architecture Diagrams

## Purpose

This diagram family contains architecture blueprints, capability maps, integration views, and target-state models for enterprise transformation.
Use it when the story needs to connect strategy, current state, and target state in one place.

## Typical Diagrams

- capability maps
- target-state architectures
- integration patterns
- modernization roadmaps
- governance reference views
- operating model views
- platform boundary diagrams
- transformation sequencing maps

## Use

- link diagrams to the architecture blueprints repository
- keep diagrams aligned with roadmap decisions
- make sure the visuals support both executive and implementation audiences
