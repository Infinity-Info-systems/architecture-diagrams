# Source Files

## Purpose

Keep editable source files for every diagram here.
This is the working area where the visual structure should be maintained before exports are generated.

## Expected File Types

- draw.io
- Visio
- editable vector files
- any future source format that can round-trip cleanly into export assets

## Use

- keep source and export names aligned
- update source first, then export
- include a short note in the source file name if the diagram is tied to a specific repo or publication
