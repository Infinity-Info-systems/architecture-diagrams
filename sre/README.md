# SRE Diagrams

## Purpose

This family holds SRE operating model diagrams, reliability workflows, and incident response views.
It should help teams explain how reliability work is governed, measured, and improved over time.

## Typical Diagrams

- SRE operating model
- incident lifecycle
- SLO governance loop
- error budget flow
- on-call flow
- post-incident learning loop
- service ownership map

## Use

- pair visuals with the SRE reliability models repository
- use them for reliability reviews and training
- keep the visuals strong enough for executive reviews and practical enough for engineering use

## Use Guidance

Use this family when the visual needs to explain reliability ownership, incident flow, or learning loops.
