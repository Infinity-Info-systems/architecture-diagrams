# FinOps Diagrams

## Purpose

This family holds cloud cost, allocation, optimization, and governance visuals.
Use it to explain cost ownership and decision-making across cloud teams.

## Typical Diagrams

- cost governance flow
- rightsizing loop
- allocation model
- executive cost view
- spend-to-value map
- budget review flow
- savings realization loop

## Use

- pair visuals with the FinOps governance repository
- use them in budget and optimization reviews
- make cost narratives accessible to both finance and engineering audiences

## Use Guidance

Use this family when the visual needs to explain cost ownership, optimization, or savings validation.
